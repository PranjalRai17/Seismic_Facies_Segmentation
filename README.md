# Seismic Facies Segmentation (Netherlands F3)

This project focuses on seismic facies classification using deep learning techniques on the Netherlands F3 dataset.

## Objectives
- Understand seismic data
- Perform facies segmentation
- Build deep learning models (U-Net)

## Dataset
Netherlands F3 dataset (from Zenodo)

## Structure
- data/ → raw + processed data
- src/ → core ML code
- notebooks/ → experiments
- outputs/ → results
